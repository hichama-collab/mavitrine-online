# mavitrine-online

Repo local de preparation pour le site agence `Ma Vitrine Online`.

## Structure

- `site/`
  code du site public
- `content/`
  textes, offres, FAQ, argumentaires
- `assets-source/`
  logos, visuels, branding source
- `notes/`
  roadmap, offres, positionnement, pages a faire
